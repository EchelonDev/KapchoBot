Kapcho Bot

A Discord bot for osu! discord servers.

Get Kapcho

If you want Kapcho Bot in your own server, you only need to send a pm to bot itself.

Just join Offical Kapcho Bot Server with this, then send a pm to bot. Then you can either write "!help" to the channel or pm to bot with "!join-server [discord invite link url]".

Commands

Command Description Usage
!msg Sends a message to an offline person for him to see when he becomes online. <user> <message>
!zar Rolls a random number to upper limit from 0. <limit>
!join-server Make bot join a server with invite link. <invite link>
!bye Says bye to user or parameter <user>
!ping Responds by pong !
